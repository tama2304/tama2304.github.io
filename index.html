Random Quote Generator

For this project, we used HTML and JavaScript to randomly generate a quote from a set of seven. JavaScript picks one of these quotes from the array using a Math.random method inside of the click function of the button. I also used CSS to style the page with different fonts and colors to make it look more visually appealing.

Photo Gallery Project

For this project, we had to use HTML, CSS, and JavaScript to create a photo gallery with a theme of our choice. We needed to add three images from the start, and then use JavaScript to make it so you can add more images using inputs and a click function. For my photo gallery, I chose a theme of video game consoles and put the Nintendo Switch, Playstation 5, and Xbox Series X as the first three images. I also added captions underneath all the images and used CSS to make the images the same size, as well as to add some appealing colors and fonts to the page.

Character Quiz

For this project, we had to use HTML, CSS, and JavaScript to create a character quiz using a theme of our choice. We were required to have at least two questions, one being answered with a number, and four possible answers. For my quiz I chose Pokémon as a theme, and asked the age of the quiz taker as well as if they like cool or cute Pokémon. Depending on their answers, they are Jigglypuff, Pikachu, Charizard, or Mewtwo. The JavaScript determines which Pokémon you are using conditional statements. The combination of conditional statements you input determines which Pokémon you are.
